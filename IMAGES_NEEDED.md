# Benötigte Bilder

Um die Webseite vollständig anzuzeigen, müssen folgende Bilder hinzugefügt werden:

## Porträtfoto

**Pfad:** `/public/hugo-ramnek.jpg`

Das Porträtfoto von Hugo Ramnek, das du hochgeladen hast, sollte hier gespeichert werden.
- Format: JPG
- Empfohlene Größe: mindestens 800x800px (quadratisch)
- Das Bild wird in der Hero-Sektion angezeigt

## Buchcover

Alle Buchcover sollten im Ordner `/public/covers/` gespeichert werden:

1. `/public/covers/die-laengste-nacht.jpg`
2. `/public/covers/die-schneekugel.jpg`
3. `/public/covers/das-letzte-von-leopold.jpg`
4. `/public/covers/meine-ge-ge-generation.jpg`
5. `/public/covers/der-letzte-badegast.jpg`
6. `/public/covers/kettenkarussell.jpg`
7. `/public/covers/momentum.jpg`
8. `/public/covers/wo-kommen-die-worte-her.jpg`
9. `/public/covers/gluecksvogel.jpg`

**Empfohlene Spezifikationen:**
- Format: JPG oder PNG
- Seitenverhältnis: 3:4 (Hochformat)
- Empfohlene Größe: 600x800px oder höher
- Optimiert für Web (max. 200KB pro Bild)

## Platzhalter verwenden

Falls die echten Buchcover noch nicht verfügbar sind, können vorübergehend Platzhalter-Bilder verwendet werden. Diese können mit einem Service wie:
- https://placeholder.com
- https://via.placeholder.com
- oder eigene Grafiken erstellt werden

## Bilder optimieren

Vor dem Hochladen sollten alle Bilder für das Web optimiert werden:
- Verwende Tools wie TinyPNG, ImageOptim oder Squoosh
- Ziel: gute Qualität bei kleiner Dateigröße
- Das verbessert die Ladegeschwindigkeit der Webseite
